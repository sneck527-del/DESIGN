## 二、报价生成核心逻辑

### 2.1 工程量计算基础

根据房间属性自动计算：

```javascript
// 基础工程量
地面面积 (fa) = room.area
顶面面积 (ca) = room.area
墙面面积 (wa) = room.perimeter × room.height - 门面积 - 窗面积
踢脚线长度 (sk) = room.perimeter - 门宽度总和
门数量 = room.doors.length
门套周长 (dp) = Σ(门宽 + 门高) × 2
窗套周长 (wp) = Σ(窗宽 + 窗高) × 2
窗口总宽 (ww) = Σ(窗宽)
窗台板数量 (ws) = room.windows.length
```

### 2.2 材料筛选规则

```javascript
// 筛选逻辑
1. 排除 calcType === 'building_area' 的材料（按建筑面积计算的材料单独处理）
2. 排除 '窗帘盒制作'（特殊处理，有吊顶时才生成）
3. 如果材料 spaceTypeFilter 为空数组 []，则该材料适用于所有空间类型
4. 如果材料 spaceTypeFilter 包含房间的空间类型ID，则该材料适用于此空间
```

### 2.3 材料排除规则（根据装修材料选择）

**地面材料排除：**
- 选择 `flooring`（木地板）：排除 m40(地砖密缝)、m41(地砖常规)、m44(大理石)、m46(其他)
- 选择 `ceramic_tile`（瓷砖）：排除 m43(木地板)、m44(大理石)
- 选择 `marble`（大理石）：排除 m40、m41、m43、m46

**墙面材料排除：**
- 选择 `paint`（涂料）：排除 m42(墙面砖)、m47(石材安装)、m71/m72/m73(墙面石材)
- 选择 `wallpaper`（墙纸）：排除 m42、m47、m71/m72/m73
- 选择 `wall_tile`（墙砖）：排除 m71/m72/m73

**顶面材料排除：**
- 选择 `none`（无吊顶）：排除 m50(石膏板吊顶)、m51(石膏板吊顶)、m80(铝扣板)、m81(其他吊顶)

## 七、报价生成流程

```
1. 用户输入房间信息
   ↓
2. 标准化房间名称和楼层名称
   ↓
3. 根据空间类型筛选材料 (spaceTypeFilter)
   ↓
4. 根据用户选择的装修材料排除不适用的材料
   ↓
5. 计算各材料的工程量 (calcType)
   ↓
6. 生成报价项目列表
   ↓
7. 添加自定义项目（如有）
   ↓
8. 添加设备产品（如有）
   ↓
9. 添加背景墙项目（如有）
   ↓
10. 渲染报价表格
```

## 八、维护说明

### 8.1 添加新的空间类型

1. 在 `materials-data.js` 的 `DEFAULT_SPACE_TYPES` 中添加：
```javascript
{id:'st_new_type',name:'新空间',icon:'🏠'}
```

2. 在本文件中添加对应的报价项目明细章节

3. 根据需要更新材料的 `spaceTypeFilter` 配置

### 8.2 修改材料的适用空间

在 `materials-data.js` 中修改对应材料的 `spaceTypeFilter`：

```javascript
// 适用于所有空间
spaceTypeFilter: []

// 仅适用于卫生间
spaceTypeFilter: ['st_bathroom']

// 适用于卫生间和厨房
spaceTypeFilter: ['st_bathroom', 'st_kitchen']
```

### 8.3 添加新的材料

在 `materials-data.js` 的 `DEFAULT_MATERIALS` 中添加：

```javascript
{
  id: 'm_new',
  name: '新材料名称',
  brand: '品牌',
  unit: '单位',
  prices: {luxury: 0, premium: 0, standard: 0},
  description: '说明',
  category: '类别',
  calcType: '计算类型',
  spaceTypeFilter: ['适用空间ID'],
  processDetail: {
    luxury: '豪华方案工序',
    premium: '优质方案工序',
    standard: '标准方案工序'
  }
}
```

### 8.4 修改报价计算规则

1. 修改 `calcType` 改变工程量计算方式
2. 修改 `spaceTypeFilter` 改变适用空间
3. 修改 `prices` 改变单价
