## 二、功能模块设计

### 模块1：房间信息录入
- 房间名称（如：客厅、主卧、厨房…）
- 长、宽、高
- 门数量及尺寸（宽×高）
- 窗数量及尺寸（宽×高）
- 可添加多个房间

### 模块2：自动计算工程量
根据输入的尺寸，自动计算：
- **地面面积** = 长 × 宽
- **顶面面积** = 长 × 宽（扣除吊顶异形另算）
- **墙面面积** = (长 + 宽) × 2 × 高 - 门窗面积
- **门窗面积** = 各门窗面积之和
- **踢脚线长度** = (长 + 宽) × 2 - 门洞宽

### 模块3：报价明细表
| 列名 | 说明 |
|------|------|
| 项目 | 如：墙面乳胶漆、地砖铺贴等 |
| 数量 | 自动根据工程量计算 |
| 单位 | ㎡、m、项等 |
| 单价 | 可编辑的市场价 |
| 价格 | 数量 × 单价（自动计算） |
| 工艺/材料/设备说明 | 施工工艺和材料品牌规格 |

### 模块4：汇总与附加
- 各房间小计
- **管理费**（通常为总价的 5%-10%）
- **税金**（通常 3%-6%）
- **垃圾清运费**
- **成品保护费**
- **总价合计**

## ✅ 已完成功能总览

### 1. 客户信息
- 客户姓名、联系电话、装修地址、报价日期、建筑面积（用于水电计价）

### 2. 房间信息录入
- 支持 11 种房间类型：客厅、餐厅、卧室、厨房、卫生间、阳台、玄关、走廊、书房、储物间、其他
- 输入长/宽/高，自动计算：**地面面积、墙面面积、顶面面积、踢脚线长度**
- 动态添加多扇门（可设置"有门/无门"区分门套和垭口）
- 动态添加多个窗户
- 支持编辑和删除房间

### 3. 三套单价方案切换
| 方案 | 特点 |
|------|------|
| **豪华型** | 冲筋找平、密缝铺贴、无主灯吊顶、进口品牌漆、实木复合门等 |
| **精装型** | 普通找平、常规铺贴、石膏板吊顶、国产品牌漆 |
| **标准型** | 基础找平、常规铺贴、强化地板、复合门 |

### 4. 报价明细表
- 按分类分组：墙面工程 → 地面工程 → 顶面工程 → 门窗工程 → 水电工程 → 安装工程 → 其他费用
- 每行显示：序号、项目、空间、数量、单位、单价、金额、工艺/材料/设备说明
- 每个分类有小计
- **水电按建筑面积单列**，不在每房间计算
- 支持添加自定义项目（可编辑数量和单价）

### 5. 费用汇总
- 各分类小计 → 工程小计 → 管理费（可调比例）→ 税金（可调比例）→ **报价总计**

### 6. 输出功能
- **PDF导出**：生成专业格式的报价单PDF（含客户信息、房间明细、报价表、条款、签字栏）
- **打印**：新窗口打印，排版整洁
- **保存/加载**：数据保存到浏览器本地存储
